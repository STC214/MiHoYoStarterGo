# MiHoYoStarterGo

基于 `Go + Wails + Vue3` 的米哈游账号切换与登录辅助工具（本地离线 OCR）。

## 当前状态

- 支持游戏：
  - `GenshinCN`（原神国服）
  - `StarRailCN`（崩坏：星穹铁道）
- `ZZZCN`（绝区零）模块已**整体下线清理**，等待后续重新梳理业务流程后再重做。

## 核心能力

- 本地账号管理（增删改、排序、加密存储密码）
- 游戏路径管理
- 运行态流程控制（启动、暂停、停止监控）
- OCR 驱动登录流程识别与自动输入
- 登录成功后读取注册表 Token 并写回配置

## 安全与数据

- 所有识别与流程均在本机执行，不依赖在线接口
- 配置文件为项目根目录 `config.json`
- 调试截图会写入项目目录（按当前逻辑）

## 项目结构（主要）

- 后端入口：`app.go`
- 业务层：`app_logic/`
- 核心逻辑：`logic/`
- 前端：`frontend/`

## 本地运行

1. 安装依赖（Go、Node、Wails 环境）
2. 前端开发：
   - `cd frontend`
   - `npm install`
3. 项目根目录运行：
   - `wails dev`

## 构建

- 前端构建：
  - `cd frontend`
  - `npm run build`
- Go 编译检查（项目根目录）：
  - `go build ./...`
- Wails 打包（项目根目录）：
  - `wails build`
